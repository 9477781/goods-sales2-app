# Antigravity Project Constitution

## MVP Scope

- コラボグッズの在庫・販売状況をリアルタイムで確認・管理できるウェブアプリケーション。
- 店舗ごとの在庫状況の表示、フィルタリング機能。
- 多言語対応（日本語・英語）。

## Tech Stack

- Frontend: React, Vite, TypeScript
- Styling: Vanilla CSS
- State Management: React Hooks
- Data: JSON/Spreadsheet (via constants.ts or external source)
